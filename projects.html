Gemini Rephraser Desktop Tool

A desktop automation tool to quickly rephrase selected text in different styles (Casual, Formal, Humorous) using Google Gemini LLM. Works globally with a hotkey and provides a simple popup UI and system tray control.

Features:

  • Select and copy text anywhere (email, browser, chat, etc.)
  • Press Ctrl+Alt+Shift+R to open a popup and choose a rephrase style
  • Get the rephrased text in a new popup with copy and close options
  • System tray icon to pause/resume or exit the tool
